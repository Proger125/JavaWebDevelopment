# WorldFlat
 ## Description
 > This platform helps people from all over the world to book or to rent an apartment. Here you can find different offers with competitive price that you can book for a day or for a month. You can place here your apartments and, rest assured, somebody will book it. Try yourself and enjoy!
 ## User groups:
 * Admin
 * User
 * Guest
 ## Use cases:
 1. Guest
    * LogIn
    * Create new account
 3. User
    * Add new offer
    * Upload photos to offer
    * View all active offers
    * Reservate flats
    * Upload user icon
    * If user reservate your offer you will be notified
 5. Admin
    * Ban | Unban users
    * Ban | Unban offers
    * View users list
    * View reservations list
    * View offers list
    * Upload admin icon
